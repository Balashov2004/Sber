# Этап 4. Визуализация и выводы

## Основной отчёт

Главный аналитический артефакт проекта:

`notebooks/final_sber_procurement_report.ipynb`

Notebook содержит методологию сбора, описание источников, устройство
PostgreSQL, SQL-запросы, гипотезы, графики, LLM и итоговые ограничения.
Все графики встроены в Notebook как сохранённые SVG-результаты и не зависят от
отображения внешних ссылок редактором.

## Визуализации

1. динамика количества и суммы по месяцам;
2. структура направлений;
3. топ-20 крупнейших закупок;
4. IT-закупки и USD/RUB;
5. строительство и ключевая ставка;
6. распределение указанных сумм;
7. доступность НМЦ и итоговой цены;
8. качество локального LLM-анализа.

Каждый блок содержит:

- наблюдение;
- интерпретацию;
- значимость;
- ограничение.

## Ключевые выводы

- максимум количества — ноябрь 2025 года, 181 процедура;
- максимум месячной суммы — февраль 2025 года, 413,6 млн руб.;
- IT и телеком выросли с 61 до 149 процедур;
- сумма IT выросла на 254,88%;
- связь суммы IT с USD/RUB не подтверждена;
- связь строительства с ключевой ставкой не подтверждена;
- указанная сумма отсутствует у 278 процедур;
- итоговая цена контракта отсутствует во всём наборе;
- Qwen пригодна для черновой разметки, но требует ручной проверки.

## Ограничение НМЦ → контракт

Распределение снижения НМЦ не рассчитывается, поскольку итоговая цена
заключённого договора отсутствует. Для расчёта нужны:

- итоговая цена;
- победитель;
- число участников;
- ссылка на протокол;
- дата договора.
